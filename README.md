# Descripcion General del Sistema
Sistema de gestion por consola para la veterinaria 'Patas y Bigotes' (Escenario 14), desarrollado integramente en Python. El software permite:
- Registrar mascotas.
- Registrar turnos.
- Registrar atenciones médicas.
- Mostrar mascotas registradas.
- Mostrar turnos registrados.
- Consultar estadísticas del sistema.
La información se almacena en archivos de texto plano.

## Tecnologías utilizadas:
- Python 3
- Archivos de texto (.txt)

## Estructura del proyecto:
- main.py
- mascotas.txt
- turnos.txt
- atenciones.txt

## Instrucciones de ejecucion:
1. Asegurarse de tener Python 3 instalado.
2. Descargar o clonar el repositorio.
3. Abrir la terminal en la carpeta del proyecto y ejecutar: python3 main.py
